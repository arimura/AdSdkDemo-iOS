# AdSdkDemo-iOS

This iOS project includes SDKs for displaying ads.

## SDK documentation
- Google Mobile Ads SDK: https://developers.google.com/ad-manager/mobile-ads-sdk/ios/quick-start
- MAX/AppLovin SDK: https://support.axon.ai/ja/max/ios/overview/integration/

## SDK header files
SDKs may include header files for their public APIs.
Header files may be located in the Headers directory inside each .xcframework.
